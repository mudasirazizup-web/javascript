# JavaScript Basics Demo

This project is a simple HTML file that demonstrates basic JavaScript concepts such as:

- Variables
- Data Types
- Console Output
- Template Literals
- Type Conversion
- `typeof` Operator
- `parseInt()` Function

---

🚀 Features
1. Console Logging

Print messages and variables using console.log().

Example:

console.log("My Name Is Mudasir");
2. Variables

Using let and const to store data.

Example:

let name = "LimNumber";
let age = 34;
const schoolgo = true;
3. Template Literals

Display variables inside strings using backticks.

Example:

console.log(`My Name is ${name} and my age is ${age}`);
4. Checking Data Types

Using typeof to check variable types.

Example:

console.log(typeof age);
console.log(typeof mymobile);
5. Type Conversion

Convert string values into numbers using parseInt().

Example:

mymobile = parseInt(mymobile);
6. NaN Example

Trying to convert a non-numeric string into a number results in NaN.

Example:

let myfathers = "abdul";
myfathers = parseInt(myfathers);

console.log(myfathers); // NaN
🖥️ How to Run
Save the code in a file named index.html
Open the file in any web browser
Open Browser Developer Console:
Right Click → Inspect
Go to the Console tab
View the output
